# Property Radar

**Property Radar** is a sophisticated, real-time interactive mapping platform designed for high-performance property visualization. It provides a seamless interface for exploring real estate projects across the Emirates, featuring advanced multi-layer spatial analysis and intuitive market data filtering.

## 🚀 Key Features

- **Interactive Spatial Visualization**: A high-performance map interface powered by Leaflet, offering fluid navigation and precise project positioning.
- **Multi-Layer Map Engines**: Switch between multiple map perspectives to suit your analysis needs:
  - **Detailed Streets**: Optimized for urban navigation.
  - **Light & Dark Modes**: Reduced eye strain and sleek aesthetics for professional environments.
  - **Satellite 3D**: High-resolution imagery provided by Esri for environmental context.
- **Advanced Market Filtering**: Instantly narrow down listings with a powerful dashboard:
  - **Location Intelligence**: Filter by Emirate (Dubai, Abu Dhabi, Sharjah, etc.).
  - **Project Status**: Tracking both **Off-Plan** and **Ready** properties.
  - **Price Tiering**: Multi-select price ranges with synchronized color-coded markers.
  - **Dynamic Categories**: Sector-specific filtering (Apartments, Villas, etc.).
  - **Bedroom Distribution**: Granular search from Studios to multi-bedroom residences.
- **Visual Intelligence**: Custom-engineered markers that change color based on price brackets, providing an immediate visual heat-map of property values.
- **Rich Interactive Insights**: Professional property popups showing real-time price, specifications, and project descriptions.
- **Performance Optimized**: Leverages **IndexedDB** for sophisticated client-side data caching, ensuring lightning-fast interactions even with large datasets.

## 🛠 Technology Stack

- **Frontend**: HTML5, CSS3 (Custom Variables, Glassmorphism, Responsive Grid)
- **Logic**: ES6+ JavaScript, jQuery
- **Mapping**: Leaflet.js
- **Data Persistence**: IndexedDB API
- **Fonts**: Inter (Google Fonts)

## 📦 Getting Started

### Prerequisites

No backend is required. The application runs entirely in the modern browser ecosystem.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/mansres/REmap.git
   ```
2. Navigate to the project directory:
   ```bash
   cd REmap
   ```
3. Open `index.html` in your preferred web browser.
